# Bear UI v1.1.4 — More Components, Clearer Docs

**Bear UI** (@forgedevstack/bear) is a React component library built on Tailwind CSS: zero config, TypeScript-first, and part of the ForgeStack ecosystem.

**What’s new in v1.1.4:**

✅ **22+ new components** — Popconfirm, Result (success/error/404), Descriptions, Anchor (scroll-spy), Affix, RingProgress, Spoiler, LoadingOverlay, Blockquote, Indicator, CheckboxCard, RadioCard, ActionIcon, Fieldset, and more.  
✅ **Docs upgrade** — Each component page now shows a **lines-of-code badge** so you can see size at a glance.  
✅ **Same stack** — Dark/light theming, custom colors and variants via BearProvider, Typography-driven text.

**Quick start:**

```bash
npm install @forgedevstack/bear
```

```tsx
import '@forgedevstack/bear/styles.css';
import { Button, Popconfirm, Result } from '@forgedevstack/bear';
```

Use **Popconfirm** for inline “Are you sure?” flows and **Result** for full-page success/error/404 screens. **CheckboxCard** and **RadioCard** are great for plan or option selection; **Anchor** gives you a table-of-contents style nav with scroll-spy.
